# snoozeyoulose

🎮 SnoozeYouLose — a cozy farm-sim × Galgame hybrid for desktop

| Category | Stack                        |
| -------- | ---------------------------- |
| Runtime  | Electron                     |
| UI       | React 19 · shadcn/ui         |
| 3D       | React Three Fiber · Three.js |
| Build    | Vite · electron-builder      |
| State    | Zustand                      |

> English | [简体中文](./README.zh.md)

## Install

Download the latest installer for your platform from [Releases](https://github.com/tlyboys/snoozeyoulose/releases/latest):

- **macOS** — `SnoozeYouLose-Mac-<version>-<arch>-Installer.dmg` (Apple Silicon / Intel), signed & notarized
- **Windows** — `SnoozeYouLose-Windows-<version>-Setup.exe`
- **Linux** — `SnoozeYouLose-Linux-<version>-x86_64.AppImage`

## Usage

### Play

Start a new game and tend your farm — plant, water, and harvest crops, then chat with NPCs through branching Galgame-style dialogue.

### Mods

A data-driven mod system lets you add crops, items, NPCs, and scripts without writing code.

### Updates

The app auto-updates from this repository's releases, using differential (blockmap) downloads.
